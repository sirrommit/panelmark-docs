# Renderer Spec Overview

The renderer specification defines what a package must implement to be a compatible panelmark renderer, what behavior is portable across all renderers, and how renderer-specific extensions should be documented.

*Full content is being migrated from `panelmark/docs/renderer-spec/overview.md`. See the documentation plan.*

- [Contract](contract.md) — normative requirements for renderer compliance
- [Portable Library](portable-library.md) — interactions and widgets required across all compliant renderers
- [Extensions](extensions.md) — how renderer-specific additions should be documented
- [Readiness](readiness.md) — how to assess and state renderer compliance level
