# panelmark-tui

The terminal renderer for the panelmark ecosystem. Provides a fullscreen blessed-powered terminal event loop, a built-in interaction and widget library, and testing utilities.

## What it provides

- fullscreen terminal event loop (blessed-based)
- modal helpers and terminal-specific runtime behavior
- built-in TUI interaction library
- built-in TUI widget library
- terminal rendering and testing utilities

## What it does not include

- web or desktop rendering
- browser event handling or WebSocket transport

## Who should use it

Application authors who want to build terminal UIs with panelmark.

## Current status

`panelmark-tui` is the most mature renderer in the ecosystem.
